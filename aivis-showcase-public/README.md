# AiVIS.biz Public Showcase (Minimal)

This is a curated public-facing showcase copy of AiVIS.biz.

## Purpose

- Demonstrate brand, positioning, and UX direction
- Show selected visual assets and messaging
- Avoid exposing proprietary audit orchestration, AI consistency logic, backend scoring internals, and private implementation methods

## Included

- Static homepage: `index.html`
- Curated assets: `assets/images/*`
- Public-only narrative and image notes

## Explicitly Excluded

- Server source code and model routing internals
- Prompting/fallback logic for output consistency
- Private middleware/business logic and DB implementation
- Secrets, keys, and operational infrastructure

## How to use

Open `index.html` directly in a browser, or host this folder as static content.
